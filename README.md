# ClousX

منصة ذكية لتوليد يوزرات انستقرام نادرة ومميزة.

## المميزات
- توليد يوزرات شبه رباعية بنسبة 50%.
- توليد يوزرات خماسية بنسبة 70%.
- نظام حماية من الاحتيال وبصمة رقمية.
- واجهة مستخدم احترافية تدعم الوضع الليلي.

## التقنيات المستخدمة
- HTML / CSS / JS (Vanilla)
- Firebase (Hosting, Realtime Database, Analytics)
- GitHub Actions (CI/CD)
